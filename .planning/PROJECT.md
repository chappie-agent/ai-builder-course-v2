# AI Builder Course Platform

## Overview

Transform the `/apps/app` SaaS boilerplate (next-forge) into a course delivery platform for AI builders.

## Goal

Build a fully functional online course platform where students can browse, enroll in, and complete AI courses. The platform sits on top of an existing Next.js 16 monorepo with auth, payments infrastructure, and a deployed marketing site.

## Context

- **Marketing site**: Already deployed at `ai-builder-course-v2.vercel.app` (`/apps/web`)
- **App shell**: `/apps/app` — Next.js 16, Clerk auth, Prisma DB, Tailwind CSS
- **Monorepo**: next-forge with Turborepo, Liveblocks, design system packages
- **Stack**: Next.js 16, Clerk, Prisma, Tailwind, shadcn/ui, TypeScript
